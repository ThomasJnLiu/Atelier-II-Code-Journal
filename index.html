Href vs src

As someone who gets href and src mixed up frequently, this post will be dedicated to differentiating the two html attributes for my own clarity.

Href (Hypertext Reference) is used when specifying the location of a resource on the web, such as another file or webpage.

Some uses of href:

<!--adding css to a document--!>
<link rel = "stylesheet" href = "styles.css">
<!--linking to another site--!>
<a href = "example.html">Click here</a>

Src (source) is used when embedding a resource in the document. It replaces the tag it is an attribute of with the resource when the resource is loaded.

Some uses of src:

<!--adding an image--!>
<img src = "image.jpg">
<!--adding a script--!>
<script src="script.js"></script>
